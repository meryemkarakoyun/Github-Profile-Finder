# 🔍 GitHub Profile Finder

A lightweight GitHub user search application built with **HTML**, **CSS**, and **Vanilla JavaScript**. Users can search by username to view profile details, follower stats, and the latest repositories — all powered by the **GitHub REST API**.

## 🚀 Features

- 🔎 **User Search** – Search any GitHub user by username via a simple search form.
- 👤 **Profile Card** – Displays avatar, full name, username, and bio.
- 📊 **User Stats** – Shows followers, following, and public repository count.
- 📁 **Top Repositories** – Lists the 3 most recent repositories with direct links.
- ⚠️ **Error Handling** – User-friendly messages when a profile is not found or repos fail to load.
- 🌙 **Dark Theme UI** – GitHub-inspired dark layout with a clean card design.
